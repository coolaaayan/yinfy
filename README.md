# yinfy
Link 2 computers to get 1 big one. 
How to Link
Install: Run Yinfy_Setup.exe on both computers.

Follower: Open Yinfy on PC 2 and click "Become Follower".

Leader: Open Yinfy on PC 1, click "Become Leader", and enter PC 2's IP.

Boost: Once the bars on the dashboard turn green, import your ISO and hit "Boot Cluster".
